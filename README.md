# Payback Card Manager

Eine Android App zum Speichern und Anzeigen von Payback-Karten Barcodes.

## Funktionen
- Payback-Karten mit Namen und Nummer speichern
- Barcode/QR-Code Generierung basierend auf Kartennummer
- Vollbild-Anzeige der Barcodes
- Einfache Benutzeroberfläche
- Daten werden lokal auf dem Gerät gespeichert

## Installation
1. Lade die APK von GitHub Actions herunter
2. Aktiviere "Installation aus unbekannten Quellen" auf deinem Android-Gerät
3. Installiere die APK
4. App öffnen und Karten hinzufügen

## Verwendung
1. App öffnen
2. Auf "+" klicken um eine Karte hinzuzufügen
3. Kartenname und Nummer eingeben
4. Auf "Karte speichern" klicken
5. Auf Karte in der Liste tippen um Barcode im Vollbild anzuzeigen
6. Beim Kassierer scannen lassen

## Build
Die APK wird automatisch bei jedem Push zu GitHub erstellt.
